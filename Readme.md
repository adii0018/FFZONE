## 🌌 Overview

<img align="right" src="https://media.giphy.com/media/l0HlBO7eyXzSZkJri/giphy.gif" width="260" />

> **FFZONE** is a high-performance, full-stack tournament management platform built with a **Cyber-Neon** aesthetic — where speed meets precision.

It brings the thrill of professional esports to every player, featuring:

- 🔴 Real-time match updates & live countdowns
- 💜 Secure Razorpay payment system
- 🔵 Sleek immersive Cyber-Neon UI
- ⚡ Zero-latency notifications across all devices
- 🏆 Full bracket & tournament management
- 🔍 Team finder to build your dream squad

> [!IMPORTANT]
> This project follows the **Tactical Kineticism** design system — built for elite players, forged for the battlefield.

## 🚀 Quick Start

<img align="left" src="https://media.giphy.com/media/LmNwrBhejkK9EFP504/giphy.gif" width="190" />

### 📋 Prerequisites

```
Node.js  ≥ 16
Python   ≥ 3.8
Git
```

### ⚡ One-Click Start (Windows)

> Double-click karo aur game mein ho jao — no terminal needed!

| Action | Script |
|:---|:---|
| 🆕 **First Time Setup** | `run_project.bat` |
| ▶️ **Daily Start** | `start_servers.bat` |
| ⏹️ **Safe Stop** | `stop_servers.bat` |
